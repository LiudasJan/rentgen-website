ChatGPT API – breaking the magic box

Tested with Rentgen · October 2025

We’re surrounded by AI tools that promise magic. But when you actually test the APIs behind them, things start to look very human.

What was tested

A real authenticated request taken directly from the browser. No guessing. No reverse engineering.

What Rentgen found

  • Missing Cache-Control headers
  • Loose input validation
  • 500 errors on malformed payloads

Final thoughts

Even the most advanced platforms ship imperfect APIs. Tools don’t replace judgment — they expose reality.
